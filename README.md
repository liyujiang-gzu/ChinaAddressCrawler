# ChinaAddressCrawler

中国行政区划数据（2022年）采集及转换。

> SQL格式数据来源于国家民政局，包括省级、地级、县级。
> 
> JSON格式数据来源于国家统计局，包括省级、地级、县级、乡级、村级。
